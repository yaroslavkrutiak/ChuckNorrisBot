## Description

This [bot](https://t.me/node_task_bot) manages to spread jokes about Chuck Norris

## Installation

```bash
$ cd./source/nest-bot-task

$ npm install
```

## Running the app

Don't forget to enter the working directory ```cd./source/nest-bot-task```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
